## Requirements

[npm](https://www.npmjs.com/)

## Installation

```bash
git clone git@github.com:dvd-z/blender.git
cd blender/auth
npm install
cd ../client
npm install
cd ..
node auth/implicit_grant/app.js
```

Open a second terminal.

```bash
cd client
npm start
```
