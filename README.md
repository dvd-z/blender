# shuffly

## WORK IN PROGRESS

## Requirements

- [npm](https://www.npmjs.com/)

## Installation

```bash
git clone https://github.com/dvd-z/shuffly.git
cd shuffly/auth
npm install
cd ../client
npm install
cd ..
node auth/implicit_grant/app.js
```

Open a second terminal.

```bash
cd client
npm start
```
